# Bisera Chadlovska — official website

A bilingual static website for conductor Bisera Chadlovska.

- English: `/`
- Macedonian: `/mk/`
- Static HTML, CSS and JavaScript
- Designed for GitHub Pages

The photography in `assets/images` is used with permission from Bisera Chadlovska's existing personal website. It is not licensed for reuse outside this project.

## Publishing

GitHub Pages can publish the repository directly from the `main` branch and the repository root. No build command or package installation is required.
